# 📅 Journal Quotidien — Vanzon Explorer

Ce dossier contient le journal des avancées quotidiennes de Jules.
Chaque fichier = une journée. Format : `YYYY-MM-DD.md`

## Objectif
Donner un contexte cumulatif au **Directeur Stratégique** (agent IA) et à Jules lui-même pour :
- Mesurer la vélocité réelle du projet (pas seulement les plans)
- Identifier les patterns (qu'est-ce qui fonctionne, qu'est-ce qui traîne)
- Contextualiser les décisions du lendemain

## Structure de chaque entrée
- **Développement** — code, features déployées
- **Acquisition** — prospection, contacts, partenariats
- **Distribution** — annuaires, plateformes, SEO
- **Résultats** — métriques concrètes, leads, réponses
- **Blocages** — ce qui n'a pas avancé et pourquoi
- **Demain** — priorités du lendemain

## Convention
- Toujours noter les chiffres concrets (X messages envoyés, X réponses, etc.)
- Un lead "chaud" = personne ayant répondu positivement en DM
- Un lead "froid" = demande envoyée sans réponse
