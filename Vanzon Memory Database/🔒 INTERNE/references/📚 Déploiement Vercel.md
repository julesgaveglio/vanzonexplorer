## Déploiement

Le repo GitHub `julesgaveglio/vanzonexplorer` est connecté à Vercel.

**Chaque `git push origin main` déclenche automatiquement un déploiement sur Vercel.**

Il n'y a rien d'autre à faire. Pas de commande manuelle, pas de CLI, pas de question à poser.

URL de production : https://vanzonexplorer.com

## Processus de déploiement

1. `git push origin main` → déploiement automatique Vercel
2. C'est tout.

## À NE PAS FAIRE

- Ne jamais demander "est-ce que le site est sur Vercel ?"
- Ne jamais proposer d'options de déploiement
- Ne jamais demander de configurer quoi que ce soit
