# Music Library

## Catégories

### 🎵 Chill / Lofi
Ambiance détendue, café du matin, intérieur cozy.
Dossier : `public/reel-music/chill/`

### 🚀 Upbeat / Energique
Transitions rapides, before/after, packing, road trip dynamique.
Dossier : `public/reel-music/upbeat/`

### 🎬 Cinematic
Plans larges, paysages, golden hour, moments épiques.
Dossier : `public/reel-music/cinematic/`

### 🌊 Adventure
Driving shots, exploration, découverte de spots.
Dossier : `public/reel-music/adventure/`

### 🍃 Nature / ASMR
Sons naturels, ambiance, pas de musique forte.
Dossier : `public/reel-music/nature/`

## Comment ajouter une musique
1. Télécharger depuis Pixabay Music ou Chosic (gratuit, libre de droits)
2. Renommer en format : `mood-description.mp3` (ex: `chill-morning-coffee.mp3`)
3. Déposer dans le bon sous-dossier
4. La musique sera auto-détectée par le pipeline
